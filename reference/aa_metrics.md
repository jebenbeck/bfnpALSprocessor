# Assess the accuracy of a set of ground control points with boxplots

This function takes the output of `merge_gcps()` and provides accuracy
metrics for the differences per gcp in X,Y,Z directions per site

## Usage

``` r
aa_metrics(gcp_data, export = F, filename, output_path)
```

## Arguments

- gcp_data:

  object of class `spatial points` as generated by `merge_gcps())`.

- export:

  logical of length 1. Should the result be exported to a csv file?
  Defaults to FALSE.

- filename:

  character, optional, name of the resulting csv file

- output_path:

  character, optional, path to folder where the csv file will be
  exported to

## Value

data frame
