# Assess the accuracy of a set of ground control points with boxplots

This function takes the output of `merge_gcps()` and provides boxplots
for the differences per gcp in X,Y,Z directions per site

## Usage

``` r
aa_create_boxplots(gcp_data, export = F, filename, output_path)
```

## Arguments

- gcp_data:

  object of class `spatial points` as generated by `merge_gcps())`.

- export:

  logical of length 1. Should the result be exported to a png ? Defaults
  to FALSE.

- filename:

  character, optional, name of the resulting png file

- output_path:

  character, optional, path to folder where the file will be exported to

## Value

aa_create_boxplots: ggplot2 object
